    tensorstore.Schema.fill_value : array_like | None

    Fill value specified by the schema.

    Example

    >>> schema = ts.Schema()
>>> print(schema.fill_value)
None
>>> schema.update(fill_value=42)
>>> schema.fill_value
array(42)
