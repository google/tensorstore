    property tensorstore.Spec.dimension_units : tuple[Unit | None, ...] | None

    Effective physical units of each dimension of the domain, including any constraints implied by driver-specific options.

    Example

    >>> spec = ts.Spec({
...     'driver': 'n5',
...     'kvstore': {
...         'driver': 'memory'
...     },
...     'metadata': {
...         'units': ['nm', 'nm', 'um'],
...         'resolution': [200, 300, 1],
...     }
... })
>>> spec.dimension_units
(Unit(200, "nm"), Unit(300, "nm"), Unit(1, "um"))

    Note

    This does not perform any I/O. Only directly-specified constraints are included.
