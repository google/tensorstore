    property tensorstore.IndexTransform.input_origin : tuple[int, ...]

    Inclusive lower bound of the input domain.

    Alias for the origin property of the domain.

    Example

    >>> transform = ts.IndexTransform(input_inclusive_min=[1, 2, 3],
...                               input_shape=[3, 4, 5])
>>> transform.input_origin
(1, 2, 3)
